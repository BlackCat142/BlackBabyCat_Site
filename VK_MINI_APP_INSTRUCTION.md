# Как добавить приложение в ВКонтакте как мини-приложение

## Что уже сделано в коде:

1. **Добавлена библиотека VK Bridge** - подключена в `<head>` вашего `index.html`
2. **Инициализация VK Bridge** - добавлен скрипт инициализации перед закрывающим тегом `</body>`
3. **Настройка темы** - приложение будет адаптироваться к теме пользователя ВКонтакте

## Пошаговая инструкция по созданию мини-приложения ВКонтакте:

### Шаг 1: Создайте приложение ВКонтакте

1. Перейдите на [VK Developers](https://dev.vk.com/)
2. Войдите под своим аккаунтом ВКонтакте
3. Нажмите **"Мои приложения"** → **"Создать"**
4. Выберите тип приложения: **"Mini App"** (Мини-приложение)
5. Заполните информацию:
   - **Название**: TheWitcheryCat
   - **Описание**: Официальный сайт музыканта из Норильска
   - **URL сайта**: https://TheWitcheryCat.ru
   - **Базовый домен**: TheWitcheryCat.ru

### Шаг 2: Настройте Mini App

После создания приложения перейдите в раздел **"Настройки"** → **"Mini App"**:

1. **URL мини-приложения**: Укажите полный URL вашего сайта
   ```
   https://TheWitcheryCat.ru/index.html
   ```

2. **URL для мобильной версии**: Тот же URL
   ```
   https://TheWitcheryCat.ru/index.html
   ```

3. **Разрешения**: Выберите необходимые разрешения (обычно достаточно базовых):
   - `user_info` - информация о пользователе
   - `photos` - если нужно работать с фото
   - Другие по необходимости

### Шаг 3: Добавьте иконки для мини-приложения

В разделе **"Настройки"** загрузите иконки:
- **Основная иконка**: 512x512 px (PNG)
- **Иконка для каталога**: 512x512 px (PNG)

> ⚠️ **Важно**: В вашем `manifest.json` указаны пути к иконкам в папке `images/`, но такой папки нет в проекте. Вам нужно:
> 1. Создать папку `images/` в корне проекта
> 2. Добавить иконки `icon-192x192.png` и `icon-512x512.png`
> 3. Или обновить `manifest.json` с правильными путями к существующим иконкам в папке `icons/`

### Шаг 4: Обновите manifest.json (рекомендуется)

Для корректной работы обновите `manifest.json`:

```json
{
  "name": "TheWitcheryCat",
  "short_name": "TheWitcheryCat",
  "description": "Официальный сайт музыканта из Норильска",
  "start_url": "/index.html",
  "display": "standalone",
  "background_color": "#0f0f0f",
  "theme_color": "#8b5cf6",
  "icons": [
    {
      "src": "icons/favicon.svg",
      "sizes": "any",
      "type": "image/svg+xml"
    }
  ]
}
```

### Шаг 5: Проверьте работу

1. После настройки вернитесь в раздел управления приложением
2. Нажмите **"Запустить"** или перейдите по ссылке на мини-приложение
3. Проверьте, что:
   - Приложение загружается
   - Все элементы отображаются корректно
   - Работает навигация
   - Музыка и видео воспроизводятся

### Шаг 6: Публикация в каталоге (опционально)

Если хотите, чтобы ваше мини-приложение было доступно всем пользователям ВКонтакте:

1. Перейдите в раздел **"Модерация"**
2. Заполните информацию для публикации
3. Отправьте на модерацию
4. Дождитесь одобрения (обычно 1-3 дня)

## Дополнительные возможности VK Bridge:

### Получение информации о пользователе:
```javascript
vkBridge.send('VKWebAppGetUserInfo')
  .then(data => console.log('Пользователь:', data))
  .catch(error => console.error('Ошибка:', error));
```

### Поделиться записью:
```javascript
vkBridge.send('VKWebAppShowWallPost', {
  message: 'Слушайте TheWitcheryCat!',
  attachments: 'url=https://TheWitcheryCat.ru'
});
```

### Открыть ссылку в браузере:
```javascript
vkBridge.send('VKWebAppOpenLink', {
  link: 'https://TheWitcheryCat.ru/moon.html'
});
```
