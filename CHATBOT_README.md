# 🤖 Умный чат-бот для TheWitcheryCat

## Описание

Виртуальный ассистент, работающий полностью в браузере без использования API. 
Основан на библиотеке **Transformers.js** от Hugging Face.

## Возможности

### 🎯 Распознавание намерений
Бот понимает следующие типы запросов:
- **Приветствия** — дружеское приветствие пользователя
- **Об артисте** — информация о TheWitcheryCat, биография, город
- **Треки и музыка** — список треков, ссылки на прослушивание
- **Соцсети** — ссылки на ВКонтакте, Telegram, YouTube, Яндекс.Музыку
- **Лунный календарь** — информация о календаре с рекомендациями
- **Благодарности** — ответы на "спасибо"
- **Прощания** — дружеское прощание

### 💬 Дружеский стиль общения
- Использование эмодзи для создания теплой атмосферы 🎵✨
- Разнообразные варианты ответов (не повторяется)
- Поддержка контекста диалога

### 🎨 Адаптивный дизайн
- Кнопка в углу экрана с анимацией
- Полноценное окно чата
- Адаптация под мобильные устройства
- Темная/светлая тема (автоматически)

## Технические детали

### Архитектура
```
chatbot.js
├── WitcheryBot (класс)
│   ├── knowledgeBase — база знаний об артисте
│   ├── friendlyResponses — варианты дружеских ответов
│   ├── init() — загрузка модели Transformers.js
│   ├── analyzeIntent() — анализ намерения пользователя
│   ├── generateResponse() — генерация ответа
│   └── handleGeneralQuery() — обработка общих запросов
└── initChatWidget() — создание UI виджета
```

### Используемые технологии
- **Transformers.js** — ML в браузере
- **Vanilla JavaScript** — без зависимостей
- **CSS Variables** — интеграция с дизайн-системой сайта

### Интеграция с сайтом
1. Автоматическая загрузка после загрузки страницы
2. Использование данных из `config.js`
3. Синхронизация со стилями сайта через CSS переменные

## Как это работает

### 1. Загрузка модели
```javascript
const { pipeline } = await import(
  'https://cdn.jsdelivr.net/npm/@xenova/transformers@2.14.0'
);
this.classifier = await pipeline(
  'text-classification', 
  'Xenova/distilbert-base-uncased-finetuned-sst-2-english'
);
```

### 2. Анализ намерения
Бот использует комбинацию:
- Ключевых слов для быстрой классификации
- ML модели для понимания контекста (опционально)

### 3. Генерация ответа
- Шаблоны ответов для частых вопросов
- Динамическая подстановка данных из CONFIG
- Случайный выбор варианта для естественности

## Преимущества решения

✅ **Работает без API** — все вычисления в браузере  
✅ **Бесплатно** — нет затрат на сервера и API ключи  
✅ **Конфиденциально** — данные не отправляются на сторонние сервера  
✅ **Быстро** — после первой загрузки модель кэшируется  
✅ **Оффлайн** — работает без интернета после загрузки  
✅ **Легко обновлять** — база знаний в одном файле
